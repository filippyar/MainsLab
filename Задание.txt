## Описание задачи

Загрузить исходные данные в локальную базу данных MS SQL Server, предобработать, создать отчет в Power BI на основании таблиц из базы данных. Определить оптимальную для данной задачи структуру БД и версии ПО кандидат может самостоятельно.

Входные данные: ссылка на файл с данными. Клиент – страховая компания, - предоставил данные о посещении клиник за последние два года. На основании этих данных он хочет получить отчет в Power BI.

## Описание входных данных

- service_date – дата оказания услуги;
- l_code – код лечебного учреждения;
- service_amount – общая стоимость оказанных услуг;
- service_number – количество оказанных услуг;
- insured – идентификатор застрахованного;
- date_of_birth – дата рождения застрахованного.

Данные можно скачать по [ссылке](https://disk.yandex.ru/d/MSgzwdMZto0McQ)

## Требования к результату

**Клиенту важно узнать из отчета**

- Изменение половозрастной структуры застрахованных по сравнению с аналогичным периодом год назад.
- Изменение средней стоимости услуги по сравнению с аналогичным периодом год назад по топ 10 по посещаемости лечебных учреждений.
- Изменение среднего количества визитов на застрахованного в сравнении с аналогичным периодом год назад. Визитом считаем последовательность услуг, оказанных одному застрахованному в одном лечебном учреждении в один день.
- Изменение среднего количества кейсов на застрахованного в сравнении с аналогичным периодом год назад. Кейсом считаем последовательность услуг, оказанных одному застрахованному в одном лечебном учреждении, с разрывом не более 14 дней.

**Клиенту важно видеть данные в разрезах:**

- По полу: 1-М, 2-Ж.
- По возрастной группе: 0-18, 18-40, 40-60, 60-80, 80+.
- По периоду.

Результат к проверке: скрипт по загрузке и предобработке данных, отчет в .pbix формате, краткое описание всех преобразований данных.

Технологии: обязательно T-SQL, Power BI. По желанию: Python/любой другой язык программирования.
